# shopping-list-electron-app

> An Electron project by [Traversy Media](https://www.youtube.com/channel/UC29ju8bIPH5as8OGnQzwJyA) from the video [Build an Electron App in Under 60 Minutes](https://www.youtube.com/watch?v=kN1Czs0m1SU&t).

## Build Setup

``` bash
# install dependencies
npm install

# start electron in dev mode
npm run start

# build for production (windows)
npm run package-

# build for production (mac)
npm run package-mac

# build for production (linux)
npm run package-win

To build production, head over to main.js and uncomment the line under SET ENV.
